Task

1. How to set justify property in the paragraph elements?

Use the text-align: justify; CSS property.

Example: 
p {
  text-align: justify;
}

This paragraph text is justified. It is aligned evenly along both the left and right margins.

2. How to add shadow to text?

Use the text-shadow CSS property.

Example: 
p {
  text-shadow: 2px 2px 5px rgba(0,0,0,0.5);
}

This text has a shadow effect.

3. How to specify all the font properties in one declaration?

Use the font shorthand property:

Example: 
p {
  font: italic bold 16px/1.5 Arial, sans-serif;
}

This text uses the font shorthand property.
